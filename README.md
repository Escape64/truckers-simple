# Простые дальнобойщики

Медитативный инди-симулятор дальнобойщика. Осенние российские дороги, красивые рассветы и грозы. Никакой спешки — просто едь и кайфуй.

## Стек

- **Движок:** Godot 4 (Forward+)
- **Язык:** GDScript
- **3D-ассеты:** Kenney.nl (CC0)
- **Платформа:** PC (Windows), Steam

## Как запустить

1. Скачай и установи [Godot 4](https://godotengine.org/download)
2. Открой Godot → Import → выбери папку проекта
3. Нажми Play (F5)

## Структура проекта

```
scenes/       — сцены (.tscn)
scripts/      — GDScript скрипты (.gd)
assets/       — модели, текстуры, звуки
ui/           — шрифты и UI-ресурсы
```
